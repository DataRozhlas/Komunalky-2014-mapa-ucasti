## Instalace

    npm install -g LiveScript@1.2.0
    npm install
    slake deploy

Hlavní stránka je v www/index.html

Mapové dlaždice jsou generovány z SVG v adresáři data pomocí [SVG Mapperu](https://github.com/economia/svg-mapper).
